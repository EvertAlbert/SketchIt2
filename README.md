# Local Setup guide
## Requirements
- Xampp with php 8.0

## how to run project

1. Bootup Xampp
2. Fill in .env
3. Start apache and mysql
4. Create database in xampp and name db 'sketchit2'
5. Run migrations
6. To run laravel project use "php artisan serve"
7. On another command line run "php artisan ws:run", this is the chat websocket
